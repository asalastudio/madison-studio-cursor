## 🔤 TYPOGRAPHY

- **Display/Logo**: Space Grotesk or Outfit — geometric, modern, aerospace
- **Headers**: Space Grotesk 600 weight
- **Body**: Inter or DM Sans 400-500 weight
- **Monospace**: JetBrains Mono — timecode, technical data
- **Micro Labels**: 9-10px uppercase with 1-2px letter-spacing

## ✨ SIGNATURE VISUAL ELEMENTS

1. **Floating Particles** — tiny dots/debris near interactive elements suggesting zero-gravity
2. **Cyan Glow** — soft outer glow on active/focused elements
3. **Motion Trails** — blur effects on playhead suggesting weightless movement
4. **Orbital Rings** — subtle circular design motifs echoing planetary orbits
5. **Star Field Texture** — 2-3% opacity particle noise on backgrounds
6. **Glassmorphism** — frosted glass effect on floating panels with backdrop-blur
7. **Scan Lines** — ultra-subtle CRT texture at 1-2% for retro-future feel
