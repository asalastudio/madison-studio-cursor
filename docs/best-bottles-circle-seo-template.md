# Best Bottles SEO Template: Circle 100 ml Clear 18-415 Antique Spray

## Purpose

This is the first product-level SEO template for Madison's Best Bottles workflow. Use it as the editorial source of truth for the Circle product group before converting the pattern into automated Product Hub copy generation.

The template is intentionally conservative: it improves search relevance and customer clarity without making unverified claims about leak resistance, regulatory status, compatibility, or exact use cases.

## Source Fields

Use structured product data first. If a field is missing, keep the copy generic rather than inventing details.

Required fields:

- `family`: Circle
- `capacity`: 100 ml
- `glassColor`: Clear
- `neckFinish`: 18-415
- `applicatorType`: Antique spray / vintage bulb sprayer
- `visualIdentity`: bulb, tassel, sprayer, cap, or closure finish
- `graceSku`
- `websiteSku`
- `shopifyProductId`
- `shopifyVariantId`
- `shopifyCdnUrl`

Optional fields:

- `caseQuantity`
- `dimensions`
- `material`
- `componentFinish`
- `secondaryVisualAttributes`
- `compatibleComponents`
- `verifiedUseCases`

## FAQ Block

### What size is the Circle clear bottle?

This product group is built around a 100 ml clear glass Circle bottle. Use verified database fields for fluid-ounce conversion when shown.

### What neck finish does it use?

The Circle 100 ml clear bottle uses an 18-415 neck finish. Compatibility should still be confirmed against the specific closure, sprayer, pump, reducer, or cap being ordered.

### What makes each variant different?

Most Circle variants share the same clear glass bottle body. The purchasable difference is usually the visually dominant component, such as the bulb sprayer color, cap finish, reducer finish, tassel detail, or ring detail.

### Should the SEO copy describe the glass as Clear?

Yes for the bottle body, but visual identity should come from the dominant purchasable component when the variant includes a sprayer, cap, reducer, pump, dropper, or other closure. For example, a clear bottle with a black bulb sprayer should be described as a clear Circle bottle with a black bulb sprayer, not as a generic clear-only variant.

### Where should product imagery come from?

Product and variant imagery should be Shopify-backed. Madison should push approved product images to Shopify, capture the Shopify CDN URL, update the Convex cache, and let the Best Bottles UI render the Convex-cached Shopify media.

### Can this template be reused for other families?

Yes. Replace the family, capacity, glass color, neck finish, and component rules while keeping the same structure: group description, variant description, SEO title, meta description, alt text, and FAQ.
